# Whats docker desktop?

A Virtual Machine

* Runs Linux
* Executes containers

**Allows acces to file system**

Tools

Docker compose, CLI

### Commands

docker images -> See our images

docker pull "image" -> Dowload images

docker image rm "name image" -> Delete images

docker create "name base image" -> Create a container

docker create --name "name container" "image" -> Create a container with name

docker start "Id of the container" -> Start a container

docker ps -> See all running containers

docker ps -a -> See all containers

docker stop "container id" -> Stop container

docker create -p27017:27017 --name monguito mongo -> Way to map a port

docker log "name container" -> see al logs

docker run -d "image" -> Easy way to create an image, container an start it

docker run --name monguito -p27017:27017 -d mongo 

doccker network ls -> See all networks

docker network create "name" -> Create a network

docker network rm "name" -> Delete a network

docker build -t "name:tag" "route" -> Create an image

docker compose up -> up containers with configurations

docker compose up --build --force-recreate -> up containers and delete cache with configurations

docker compose down -> Delete all images and container created by compose
