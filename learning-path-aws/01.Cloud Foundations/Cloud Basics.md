# What is cloud computing?


Cloud computing is the on-demand delivery of IT resources over the internet with pay-as-you-go pricing.

Hybrid deployment - it's a clod solution where you can stay the confidencial data in local for archieve the requirements of accordance and at the same time, use the resources in cloud for a dynamic scaling
